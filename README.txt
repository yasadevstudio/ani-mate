ANI-MATE - Free Anime For All

Search, stream, and download anime - no external player needed.

GETTING STARTED:
    Double-click the ANI-MATE shortcut on your desktop.
    The app will open automatically.

FEATURES:
    - Search anime by name (sub/dub)
    - In-app video player with quality switching
    - Continue watching with resume from where you left off
    - Episode tracking with watched markers
    - Auto-play next episode
    - Favorites list
    - Daily trending and airing schedule
    - Download episodes (mp4 direct, m3u8 requires ffmpeg)

KEYBOARD SHORTCUTS:
    /           Focus search bar
    Enter       Play selected episode
    Escape      Close player
    Left/Right  Previous/next episode
    Space       Play/pause
    f           Fullscreen

DOWNLOADS:
    Episodes save to: Videos\ANI-MATE\
    For HLS (.m3u8) downloads, install ffmpeg and add to PATH.

DATA:
    App data stored in: %APPDATA%\ANI-MATE\data\
